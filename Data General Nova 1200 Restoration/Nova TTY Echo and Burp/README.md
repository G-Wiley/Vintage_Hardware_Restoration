Binary files (ab files) for TTY_Echo and TTY_Burp to test the serial interface on Data General Nova computers. Instructions are provided that describe how to load the binary loader using the bootstrap loader and how to load TTY_Echo and TTY_Burp using the binary loader.

TTY_Echo functions as a simple terminal typewriter. Characters entered on the keyboard are echoed to the terminal screen or Teletype printer. The following is an example of keyboard characters output to the terminal emulator:
