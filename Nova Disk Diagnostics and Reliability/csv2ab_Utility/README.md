<h1>csv2ab Utility</h1>
<p>The csv2ab utility converts octal machine code from a spreadsheet (in comma separated value, CSV, format) to a Nova absolute binary “AB” file that can be loaded by the Nova Binary Loader. The sole purpose of csv2ab is to convert the Nova disk diagnostic, which was captured manually, to an AB file.</p>
<p>The csv2ab was written using AI. The ChatGPT input that specifies the requirements is provided here.</p>
