<h1>Nova Disk Diagnostics and Reliability</h1>
<p>This is a small project to create a Nova Absolute Binary file of the Data General Nova "Moving Head Disk Control Diagnostic" program, listing 096-000169-07, tape 095-000069-07. An assembly language listing of this diagnostic code appears on pdf pages 4 through 60 of "068-000169-13_Moving_Head_Disk_Control_Diagnostic_1977.pdf". This utility is quite convenient to use because it can run in only 4K words of memory.</p>

<p>The file "068-000169-13_Moving_Head_Disk_Control_Diagnostic_1977.pdf" is available on Bitsavers at: 
<br>   https://bitsavers.org/pdf/dg/disc/4046_4047_4049/068-000169-13_Moving_Head_Disk_Control_Diagnostic_1977.pdf
<br>In case the file moves someday, this file is in Bitsavers folder: /pdf/dg/disc/4046_4047_4049</p>

<p>The diagnostic program is available as a teletype listing but a Nova Absolute Binary file (AB file) was not found on any of the usual sources. This purpose of this project is to convert the teletype listing to an AB file so Nova enthusiasts can test their 4046/4047 disk subsystems. </p>
