<h1>ABtest files</h1>
<p>The ABtest files were created to verify that the csv2ab utility is functioning properly. A detailed description of ABtest is provided in this folder.</p>
<p>ABtest.xlsx is the source, which is also saved as ABtest.csv in Excel. The csv2ab utility converts ABtest.csv to ABtest.ab. Then the content of ABtest.ab can be verified by inspection and it can be loaded into a Nova computer using the Nova binary loader to confirm that the content was properly read into the memory of the computer.</p>
